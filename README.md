# Flipkart Sunglass Web Scraping

This project scrapes sunglass details (Brand, Description, Price) from Flipkart
using Python (requests + BeautifulSoup) and stores the result in a DataFrame.

## Tech Stack
- Python
- Requests
- BeautifulSoup
- Pandas

This project scrapes sunglass product details from Flipkart and saves the final cleaned dataset into a CSV file
